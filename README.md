# Language

- Kotlin

# Architecture

- Clean Architecture
- MVVM pattern
- SOLID principals

# Main Features

- Getting news list from API when internet is connected.
- Showing news list in offline mode when internet is disconnected. 
- Pagination list

This sample showcases:

* JetPack Compose
* Material3 Theme
* Coroutines
* Hilt
* Retrofit
* ORM (ROOM)
* Navigation Component
* Glide
* Flow
